一、输入格式：
你将接收到一个包含以下个字段的 JSON 数据：

"rule"：领域专业知识中的规则。

二、任务要求：
你要对 JSON 数据中的"rule"字段，也就是规则，请你依据规则生成一个问题，要求是简答题的形式，并给出答案。并将问题和答案以 JSON 格式返回。

以下是例子：
------
{
    "rule": "《刑法》第三百五十六条：【毒品犯罪的再犯】因走私、贩卖、运输、制造、非法持有毒品罪被判过刑，又犯本节规定之罪的，从重处罚。"
}
***
生成的问题和答案如下：
***
{
    "question": "根据《刑法》第三百五十六条的规定，何种情况下应对毒品犯罪再犯从重处罚？",
    "answer": "当行为人曾因走私、贩卖、运输、制造毒品罪或非法持有毒品罪被判过刑，又再次实施《刑法》分则第六章第七节（走私、贩卖、运输、制造毒品罪）中规定的任何毒品犯罪时，应当从重处罚。"
}
