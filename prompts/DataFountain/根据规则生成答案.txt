任务描述:
一、输入格式：
你将接收到一个包含以下个字段的 JSON 数据：

"rule"：领域专业知识中的规则。

二、任务要求：
你要对 JSON 数据中的"rule"字段，也就是规则，请你依据规则生成一个问题，要求是简答题的形式，并给出答案。并将问题和答案以 JSON 格式返回。

以下是例子：
------
{
    "rule": "必要时，组织非灾区人民政府，通过提供人力、物力、财力、智力等形式，对灾区群众生活安置、伤员救治、卫生防疫、基础设施抢修和生产恢复等开展对口支援。"
}
***
生成的问题和答案如下：
***
{
    "question": "非灾区人民政府对灾区开展对口支援的主要内容包括哪些方面？",
    "answer": "主要包括灾区群众生活安置、伤员救治、卫生防疫、基础设施抢修和生产恢复。"
}
------
